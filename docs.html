OINK Docs

OINK агрегирует контент и подключает источники Kinobox и Kinobd по Kinopoisk ID.

Для встраивания на другой сайт используйте:

<div data-kinopoisk="[ID]" id="kinobd"></div>
<script src="https://kinobd.net/js/player_.js"></script>

<div id="player"></div>
<script src="https://kinoboxtv.tatnet.app/kinobox.js"></script>
<script>kinobox('#player',{ search:{ kinopoisk:[ID] } });</script>
